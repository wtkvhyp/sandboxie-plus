## Sandboxie-Plus 是什么

Sandboxie 这个名字在老用户里早就不陌生了。它最初是一家叫 Invincea 的公司开发的 Windows 沙盒工具，后来被 Sophos 收购，再后来 2020 年干脆开源了，目前由开发者 David Xanatos 个人维护，活跃度相当高——GitHub 上能看到持续更新的提交记录。

Sandboxie-Plus 是它的增强版分支，采用全新的 Qt 界面重写，比经典版更现代、功能也更完整。目前最新版已到 v1.16.9，2026 年初刚有更新。

## 它能干什么

### 隔离运行可疑软件

这是最核心的用途。在沙盒里装上软件，运行、测试，关掉沙盒，什么都消失，不影响真实系统。对那些装上会乱写注册表、随便开机自启的流氓软件，这个方案简直是救星。

### 游戏多开、微信双开

很多人知道沙盒能多开软件，微信、QQ、游戏客户端……同一个程序同时跑多个实例，互相不干扰。这个功能在免费版里就能用，不用花一分钱。

### 访问权限精细管控

高级版可以创建"隐私沙盒"和"安全增强沙盒"，限制沙盒内软件访问文件夹、注册表、打印机、剪贴板等资源。说白了就是：你在里面跑，但你别想碰我的数据。

### 网络隔离

基于 WFP 技术实现沙盒内的网络过滤，设置出站规则，防止沙盒里的软件悄悄往外发数据。对于需要联网功能但又不放心它发了啥的软件，非常实用。

### 快照与恢复

可以给沙盒状态创建快照，随时回滚。测试某个软件对系统的影响？创建快照→安装运行→对比结果→一键恢复。这个功能对技术人员、安全研究者来说很顺手。

### AES 加密沙盒（高级加密包专属）

这是最高版本才有的功能：沙盒内容以 AES 加密存储，适合有隐私敏感需求的高级用户。
